***Computer-Vision-Robot-Hand***

## FEATURES

-Control robot hand using hand detection module by sending an array of fingers you put up

***IDE***

- Arduino IDE

***Packages installed***

- Mediapipe.

- cvzone library.

- Opencv-python.

- PySerial

- Pillow.

***Components***

- Arduino

- Robot hand intergrated using 5 servos
